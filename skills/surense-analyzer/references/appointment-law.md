## החוזר משנת 2013

**עיקרי החוזר:**
- תוקף ייפוי כוח / מינוי סוכן: **לא יותר מ-10 שנים**
- חברות הביטוח והמסלקה הפנסיונית חייבות לכבד זאת
- הסוכן מחויב לחדש את המינוי לפני תום התוקף

**ציר הזמן:**
```
2013 → כניסת החוזר לתוקף
2023 → מינויים מ-2013 פוקעים (גל ראשון)
2024 → מינויים מ-2014 פוקעים
2025 → מינויים מ-2015 פוקעים
2026 → מינויים מ-2016 פוקעים ← הגל הנוכחי
```

**מה קורה כשמינוי פג:**
```
✗ הסוכן לא יכול לקבל מידע מהחברה על הלקוח
✗ הסוכן לא יכול לבצע שינויים בפוליסה
✗ הסוכן לא יכול לגבות עמלות
✗ חשיפה משפטית לסוכן ולסוכנות
```

## תהליך חידוש דיגיטלי

```
שלב 1: פנייה ללקוח בוואטסאפ
שלב 2: זיהוי — ת.ז + OTP
שלב 3: הצגת פרטי ייפוי הכוח לאישור
שלב 4: חתימה דיגיטלית
שלב 5: שליחה למסלקה הפנסיונית
זמן: ~3 דקות ללקוח
```

## לוגיקת הטריגר בסקיל

```
appointmentDaysRemaining = (expiryDate - today).days

< 0        → appointment_expired      P0 קריטי
0 עד 90    → appointment_expiring_90d P0 דחוף
91 עד 365  → appointment_expiring_1y  P2 אזהרה

```
